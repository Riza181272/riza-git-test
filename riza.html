Buat Remote di Github.com

Step-1 Buat Reposity di Local dengan $git init, kemudian buat file dll di working tree, lakukan add&commit di local

Step-2 Buat Repository di Github.com dengan setting public nanti akan terbentuk reposity https://github.com/Github-akun/repo-name

Step-3 lakukan remote ke github dengan $git remote add origin https://github.com/Github-akun/repo-name

Step-3 lakukan $git push untuk melakukan commit local same level di remote

Make alias

alias graph="git log --all --decorate --oneline --graph"
